# Classroom Gacha

This is the Classroom Gacha application, consisting of a Python FastAPI backend and a React/Vite frontend.

## Quick Start

To start the application, simply double-click the **`start-app.bat`** file in this directory.

This script will:
1. Start the Backend server (http://127.0.0.1:8000)
2. Start the Frontend development server (http://localhost:5173)

## Project Structure

- **backend/**: Contains the Python FastAPI application and SQLite database.
- **frontend/**: Contains the React frontend application.

## Requirements

- Python 3.x
- Node.js & npm
